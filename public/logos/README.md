# Just Ship — Brand Kit

## Contents

### svg/
Vector source files — use these for web, print, and any scalable context.

| File | Usage |
|------|-------|
| mark-dark | Primary — dark circle, white sails |
| mark-blue | Accent — blue circle, white sails |
| mark-white | Inverted — white circle, blue sail |
| mark-outline-dark | Outline for light backgrounds |
| mark-outline-white | Outline for dark backgrounds |
| sails-dark | Sails only, dark |
| sails-white | Sails only, white |
| sails-blue | Sails only, blue |
| lockup-light | Logo + wordmark, light bg |
| lockup-dark | Logo + wordmark, dark bg |

### png/marks/ (512×512)
Default size exports of all mark variants.

### png/sails/
Sails-only exports without container circle.

### png/lockups/
Horizontal logo + wordmark combinations.

### png/favicons/ (16, 32px)
Dark variant for light themes, white variant for dark themes.

### png/icons/ (64–1024px)
Mark at standard icon sizes.

## Quick Reference

**Accent:** #3b82f6
